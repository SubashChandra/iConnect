A matrimony web application using web2py, which suggests best possible matches based on profile, personality and interests. Free Users will be given basic suggestions. Premium Users will be given the best and refined matched based on the additional information collected during the upgradation process. Features available to premium users include wishList, liking and spamming a profile, in-app mail to any suggested match, matches/mismatches with suggested profiles etc .

Tools Used: web2py,Bootstrap
